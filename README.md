# Quiz App

A command-line quiz application built with Python.

## Features
- Loads questions from a JSON file
- Multiple choice questions
- Shows correct answer on wrong input
- Displays final score as a fraction

## How to Run
1. Make sure Python is installed
2. Run: python quz.py
3. Answer each question with the letter (A, B, C, or D)

## Built With
- Python 3
- JSON for question storage
